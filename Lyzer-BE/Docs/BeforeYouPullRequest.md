# Before you pull request

Before you pull request please make sure you have done the following:

## Format your code

Please make sure you have formatted your code using the following command:

```bash
dotnet format
```

or 

```bash
make format
```

## Run the tests

Please make sure you have run the tests using the following command and that all tests pass:

```bash
dotnet test
```

or 

```bash
make test
```

## Update any relevant documentation

Please make sure you have updated any relevant documentation. You can find the documentation in the following locations:

- [README.md](../README.md)
- [Developer Docs](../Docs/README.md)
